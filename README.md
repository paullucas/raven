# raven
A minimal &amp; opinionated SuperCollider client

### Quick Start

Start SCSynth
<br/>
```scsynth -u 57110```

Start REPL
<br/>
```npm start```

Build a nest
<br/>
```(nest)```

Lay an egg
<br/>
```(def child (lay "test"))```

Croak
<br/>
```(croak child "freq" 120)```

Abandon nest
<br/>
```(fly)```

### Install Dependencies

```
npm install -g lumo-cljs
npm install
mkdir events
```
